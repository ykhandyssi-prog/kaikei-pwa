# 家計記録 PWA

金の出入りを記録する個人用アプリ。HTML/JS単体・外部ライブラリなし・完全オフライン。

## 構成
- `index.html` — 本体（UI・ロジック・SVG自前グラフすべて内包）
- `manifest.json` — PWA設定
- `sw.js` — Service Worker（オフラインキャッシュ）
- `icon.svg` — アプリアイコン

## 機能
- 入力：支出/収入・金額・カテゴリ・支払手段・日付・メモ・タグをフォームで
- 一覧：月ごと、日付グループ・日計表示、タップで金額変更/削除（`d`入力で削除）
- 集計：月次収支・カテゴリ別ドーナツ・残高累計の折れ線
- 設定：JSONバックアップ（書き出し/取り込み）、今月分のMarkdown書き出し、カテゴリ/支払手段の編集

## データ
- 保存先：その端末のブラウザ内（IndexedDB）。**端末ごとに別データ**。
- バックアップ：設定 → JSON書き出し。復元は取り込み（IDで重複回避してマージ）。
- Vault連携：設定 → Markdown書き出し → Monthlyノートに貼り付け。

## ローカルで動かす
```
python -m http.server 5599 --directory C:/Users/ykhan/kakei-pwa
```
→ ブラウザで http://localhost:5599 を開く。
（`index.html` を直接ダブルクリックでも動くが、PWAインストール/オフラインは http 配信時のみ有効）
